call(contract, sender, wait) → {Promise}

Executes a message call transaction, which is directly executed in the VM of the node, but never mined into the blockchain.

Parameters:
Name Type Description
contract string

Contract name
sender string

Address of the sender, if null it will use the default one
wait boolean

If set to true it will wait until the contract is mined. If not it will return the transactionHash straight away and some other process will have to monitor the status.
